# Maria Bonita

Interface de aplicativo e-commerce de moda feminina. 
<br/> 
Nele o usuário encontra lançamentos, moda fitness e moda praia.

## Tecnologias utilizadas:

- Expo
- Javascript
- Styled Components
- React Native
- Hook
- Stack Navigator
